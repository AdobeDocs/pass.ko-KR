# [!UICONTROL Premium Workflow] 개요

>[!IMPORTANT]
>
> [제품 알림](/help/authentication/product-announcements.md) 페이지에서 집계된 최신 Adobe Pass 제품 알림 및 서비스 중단 타임라인에 대한 정보를 계속 받아 보십시오.

Adobe Pass의 [!UICONTROL Premium Workflow]은(는) 인증 및 스트리밍 안정성을 개선하는 강력한 도구 모음을 제공합니다. [!UICONTROL Entitlement Service Monitoring]을(를) 통해 실시간 가시성을 확보하고, 스마트 열화 API를 통해 가동 시간을 보장하며, [!UICONTROL TempPass]을(를) 통해 유연한 액세스를 제공합니다.

MVPD 상태 서비스가 끝점 성능을 미리 추적하고 경고하는 동안 [!UICONTROL Home-Based Authentication] 및 SSO(Single Sign-On)로 사용자 경험을 개선합니다. 이러한 솔루션은 프로그래머 및 MVPD에게 실행 가능한 통찰력을 제공하고, 작업을 최적화하고, 참여를 높이고, 최대 트래픽 중에도 중단 없는 컨텐츠 전달을 보장합니다.

주요 기능:

+ [Apple SSO 개요](sso-access/partner-sso/apple-sso/apple-sso-overview.md)
+ [Amazon SSO Cookbook(REST API V2)](sso-access/platform-sso/amazon-single-sign-on/amazon-sso-cookbook-rest-api-v2.md)
+ [Roku SSO Cookbook(REST API V2)](sso-access/platform-sso/roku-single-sign-on/roku-sso-cookbook-rest-api-v2.md)
+ [홈 기반 인증(HBA)](hba-access/home-based-authentication.md)
+ [TempPass 기능](temporary-access/temp-pass-feature.md)
+ [성능 저하 기능](degraded-access/degradation-feature.md)
