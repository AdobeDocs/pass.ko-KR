# Adobe Concurrency Monitoring Services 3.6.0 릴리스 노트 {#cm-services-rns}

이 페이지에서는 이 릴리스의 새로운 기능, 변경 사항 및 알려진 문제에 대해 설명합니다.

## 릴리스 정보 {#rel-info}

버전: 3.6.0
날짜: 2025/02/04

## 릴리스 개요 {#release-overview}

* 다른 테넌트의 앱에서 실행되는 스트림의 세부 정보를 포함하도록 API를 업데이트합니다.
* API 메서드에 종료 코드를 추가합니다.
* 409 응답 및 수정 가능한 메타데이터 특성에 대한 메타데이터를 추가하여 메타데이터 API를 업데이트합니다.
