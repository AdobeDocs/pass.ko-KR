# 동시성 모니터링 소개 {#intro}

동시성 모니터링은 콘텐츠 제공자와 ID 제공자(MVPD 및 프로그래머)가 여러 애플리케이션, 디바이스 및 플랫폼에서 동시에 비디오를 스트리밍하는 데 대한 제한을 정의하고 적용할 수 있도록 하는 서비스입니다. 구독자가 동시에 볼 수 있는 스트림 수를 제어하려는 프로그래머든, 콘텐츠 파트너 전반에 사용 정책을 적용하려는 MVPD이든 필요한 도구를 제공합니다.

## 동시성 모니터링이란 무엇입니까? {#what-is-cm}

동시 모니터링 은 활성 비디오 스트리밍 세션을 실시간으로 추적 및 관리하는 중앙 집중식 서비스입니다. 이를 통해 다음을 수행할 수 있습니다.

- **구독자당 동시 스트림 제한** - 사용자가 액세스할 수 있는 동시 비디오 스트림 수를 제어합니다.
- **장치 제한 적용** - 특정 장치 유형 또는 수량으로 스트리밍 제한
- **위치 기반 정책 구현** - 지리적 위치에 따라 스트리밍 제한
- **콘텐츠 관련 규칙 만들기** - 라이브 콘텐츠와 VOD 콘텐츠에 대해 다른 제한을 적용합니다.
- **사용 패턴 모니터링** - 콘텐츠가 사용되는 방식에 대한 통찰력을 얻으십시오.

## 작동 방법 {#how-it-works}

동시성 모니터링은 간단하지만 강력한 API를 통해 작동합니다.

1. **세션 초기화** - 사용자가 콘텐츠 보기를 시작하면 응용 프로그램에서 세션을 만듭니다.
2. **정책 평가** - 서비스가 현재 사용량에 대해 정의된 정책을 평가합니다.
3. **실시간 모니터링** - 하트비트 호출은 세션을 활성 상태로 유지하고 준수 여부를 모니터링합니다.
